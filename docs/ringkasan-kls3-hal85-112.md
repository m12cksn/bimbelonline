## 1) Pengertian Kalimat Matematika
Kalimat matematika adalah kalimat angka/simbol yang mewakili masalah nyata.

Contoh:
- Cerita: “Galih punya 30 kelereng, total dengan Andi 50.”
- Kalimat matematika: `30 + ... = 50`

Inti kompetensi:
- Siswa bisa mengubah cerita -> model operasi -> jawaban.

## 2) Kalimat Matematika Penjumlahan
Bentuk utama:
- `a + x = c`
- `x + b = c`

Aturan cepat:
- Jika nilai yang hilang adalah salah satu penjumlahan, maka cari dengan pengurangan total.
- `x = c - a` atau `x = c - b`

Contoh materi:
- `30 + ... = 50` -> `... = 50 - 30 = 20`
- `8 + ... = 19` -> `... = 11`
- `... + 16 = 54` -> `... = 38`

Contoh latihan singkat:
- `18 + ... = 39` -> jawaban: `21`

## 3) Kalimat Matematika Pengurangan
Bentuk utama:
- `a - x = c`
- `x - b = c`

Aturan cepat:
- Jika pengurang yang hilang: `x = a - c`
- Jika bilangan awal yang hilang: `x = c + b`

Contoh materi:
- `80 - ... = 45` -> `... = 35`
- `32 - ... = 15` -> `... = 17`
- `... - 45 = 27` -> `... = 72`

Contoh latihan singkat:
- `56 - ... = 24` -> jawaban: `32`

## 4) Menentukan Nilai yang Belum Diketahui (Inti Bab)
Langkah standar untuk siswa:
1. Baca soal perlahan.
2. Tandai angka yang diketahui.
3. Tentukan yang ditanya (nilai hilang).
4. Susun kalimat matematika.
5. Hitung.
6. Cek balik ke kalimat semula.

Contoh lengkap:
- Soal: “Ayam mula-mula 80, sisa 45, terjual berapa?”
- Model: `80 - x = 45`
- Hitung: `x = 80 - 45 = 35`
- Cek: `80 - 35 = 45` benar.

## 5) Penerapan pada Soal Cerita & Kemungkinan Jawaban
Materi melatih dua tipe soal:
- Soal tertutup (1 jawaban pasti), misalnya durian, ayam, manggis.
- Soal terbuka (banyak jawaban benar), misalnya total tiket 95 atau tabel `+/-` dengan 5 kemungkinan.

Contoh soal terbuka:
- Total tiket = 95.
- Kemungkinan valid:
- `40 + 55 = 95`
- `41 + 54 = 95`
- `42 + 53 = 95`
- `43 + 52 = 95`
- `44 + 51 = 95`

Catatan: selama persamaan benar dan konteks masuk akal, jawaban diterima.

## 6) Asesmen di Materi Ini
Asesmen menguji:
- Isian nilai hilang campuran penjumlahan-pengurangan.
- Soal cerita kontekstual.
- Kemampuan membuat beberapa kemungkinan persamaan.

Contoh butir asesmen dari pola materi:
- `... + 48 = 84`
- `83 - ... = 38`
- `... - 31 = 19`
- Soal perpustakaan: total 96, sisa 48 -> dipinjam `48`.

## Highlight Penting (Wajib Dikuasai)
1. Membedakan operasi `+` dan `-` dari kalimat soal.
2. Menentukan posisi nilai hilang pada persamaan.
3. Memakai aturan yang tepat sesuai posisi nilai hilang.
4. Melakukan cek balik hasil ke persamaan awal.
5. Menjawab soal kontekstual dengan satuan (butir, ekor, lembar, buah).
6. Memahami bahwa beberapa soal dapat punya banyak kemungkinan jawaban benar.

## Contoh per Poin Penting
1. Penjumlahan nilai hilang: `23 + ... = 45` -> `22`
2. Penjumlahan nilai hilang di depan: `... + 36 = 68` -> `32`
3. Pengurangan pengurang hilang: `53 - ... = 32` -> `21`
4. Pengurangan bilangan awal hilang: `... - 41 = 38` -> `79`
5. Soal cerita: total buku 96, sisa 48 -> dipinjam `48`
6. Soal terbuka: cari 5 kemungkinan `a + b = 100`

## Miskonsepsi yang Sering Terjadi
1. Siswa langsung menebak tanpa membuat model persamaan.
2. Salah operasi karena tidak membaca kata kunci “jumlah/selisih/sisa”.
3. Tidak cek balik, sehingga jawaban tidak memenuhi persamaan.

## Saran Strategi Mengajar Singkat
1. Mulai dari benda konkret (kelereng/permen).
2. Ubah ke tabel (diketahui-ditanya).
3. Ubah ke kalimat matematika.
4. Hitung dan cek.
5. Akhiri dengan soal terbuka (banyak kemungkinan) untuk melatih nalar.
