2. Menafsirkan data dari piktogram.
3. Menyajikan data dalam bentuk diagram batang tegak.
4. Menyajikan data dalam bentuk diagram batang mendatar.
5. Menafsirkan data dari diagram batang.

## Ringkasan Materi Lengkap

### 1) Konsep Dasar Penyajian Data
Data dapat disajikan dari bentuk daftar/tabel ke bentuk visual agar lebih mudah:
- dibaca,
- dibandingkan,
- disimpulkan.

Bentuk penyajian yang dipelajari:
- Piktogram (diagram gambar)
- Diagram batang (tegak dan mendatar)

### 2) Piktogram
Piktogram adalah diagram yang menggunakan gambar/simbol untuk mewakili banyak data pada setiap kategori.

Komponen piktogram:
1. Judul
2. Kategori data
3. Banyak simbol pada tiap kategori
4. Keterangan simbol/skala (misalnya 1 gambar = 1 siswa atau 1 gambar = 4 siswa)

Langkah membaca piktogram:
1. Baca legenda/skala simbol.
2. Hitung banyak simbol tiap kategori.
3. Ubah ke jumlah sebenarnya (jika skala lebih dari 1).
4. Bandingkan kategori.

### 3) Piktogram dengan Skala
Tidak selalu 1 gambar = 1 data.
Jika 1 gambar mewakili beberapa data, lakukan perkalian.

Contoh konsep:
- 1 gambar = 4 siswa.
- Jika ada 3 gambar -> banyak siswa = 12.

### 4) Diagram Batang
Diagram batang menyajikan data menggunakan batang persegi panjang.

Ada dua bentuk:
1. Diagram batang tegak (vertikal): baca dari **tinggi batang**.
2. Diagram batang mendatar (horizontal): baca dari **panjang batang**.

Komponen diagram batang:
- Judul diagram
- Sumbu kategori
- Sumbu jumlah/frekuensi
- Skala angka

### 5) Membaca dan Menafsirkan Diagram Batang
Langkah membaca:
1. Perhatikan judul dan label sumbu.
2. Perhatikan skala pada sumbu jumlah.
3. Baca nilai tiap batang.
4. Tentukan data terbesar, terkecil, dan total.

### 6) Analisis Data dari Diagram
Pertanyaan umum yang dilatih:
- Kategori paling banyak.
- Kategori paling sedikit.
- Banyak data pada kategori tertentu.
- Gabungan dua kategori.
- Jumlah keseluruhan data.

### 7) Menyajikan Data dalam Berbagai Bentuk
Satu data yang sama bisa disajikan sebagai:
1. Tabel
2. Piktogram
3. Diagram batang tegak
4. Diagram batang mendatar

Inti penting: informasi harus tetap konsisten meskipun bentuk penyajian berbeda.

## Highlight Penting (Wajib Dikuasai)
1. Selalu baca **legenda/skala** terlebih dahulu.
2. Pada piktogram berskala, jumlah simbol bukan jumlah data langsung.
3. Diagram batang tegak dibaca tinggi batang, mendatar dibaca panjang batang.
4. Gunakan skala dengan benar agar tidak salah hitung.
5. Kesimpulan harus berasal dari data, bukan tebakan.

## Contoh Latihan Soal per Poin
1. Ubah data tabel alat tulis menjadi piktogram.
2. Dari piktogram, tentukan data terbanyak dan tersedikit.
3. Hitung total data dari piktogram dengan skala 1 gambar = 4.
4. Ubah data bunga menjadi diagram batang mendatar.
5. Dari diagram batang lomba, tentukan kategori dengan jumlah 3 siswa.
6. Hitung jumlah siswa pada dua kategori tertentu dari diagram batang.
7. Sajikan satu set data ke dalam 3 bentuk: piktogram, batang tegak, batang mendatar.

## Checklist Penguasaan Siswa
1. Bisa membuat piktogram dari data sederhana.
2. Bisa membaca piktogram dengan/ tanpa skala.
3. Bisa membuat diagram batang tegak.
4. Bisa membuat diagram batang mendatar.
5. Bisa menentukan nilai terbanyak/tersedikit dari diagram.
6. Bisa menghitung total data dari diagram.
7. Bisa menyajikan data yang sama dalam beberapa bentuk.

## Catatan Pengajaran
1. Mulai dari data nyata di kelas (alat tulis, warna favorit, kegiatan ekstrakurikuler).
2. Biasakan siswa menulis legenda/skala sebelum menggambar piktogram.
3. Gunakan kertas berpetak agar batang pada diagram rapi dan proporsional.
4. Latih siswa mengecek ulang: apakah total data di tabel sama dengan total di diagram.
